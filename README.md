# OnAndHome 2차 — 백엔드

## 프로젝트 개요
Spring Boot + React 기반 온라인 쇼핑몰 개선 프로젝트 백엔드.
상품, 주문, 리뷰, 관리자 기능 API 제공.

## 관련 레포
- 프론트엔드: [OnAndHomeFront](https://github.com/roach-hue/OnAndHomeFront)

## 팀 구성 및 담당
- 팀원 6명
- 본인 담당: 프론트엔드 전담 (백엔드 레포 참고용)

## 기술 스택
- Java 21
- Spring Boot 3.5.6
- Spring Security + JWT
- Spring Data JPA + MySQL
- Spring WebSocket (STOMP)
- Spring Mail (Gmail SMTP)
- OAuth2 (카카오 / 네이버 / Google)
- Thymeleaf
- Gradle
- Lombok

## 실행 방법
```bash
git clone https://github.com/roach-hue/OnAndHomeBack.git
cd OnAndHomeBack
./mvnw spring-boot:run
```
